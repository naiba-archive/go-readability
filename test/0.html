#0 野生程序员

野生程序员是指仅凭对计算机开发的兴趣进入这个行业，从前端到后台一手包揽，但各方面能力都不精通的人。野生程序员有很强大的单兵作战能力，但是在编入“正规军”之后，可能会不适应新的做事方法。—— CSDN，野生程序员的故事

他们大多数人有一个共同点：基础不扎实。比如 算法、数据结构、系统原理、计算机网络 等等，做项目的方式是入门水平 + Google，拼凑起来一个能用的项目。

2018-03-14 终身学习博客博主也是一个“野生程序员”，就在昨天晚上，一个自小就写代码的 211、985 毕业的精通各种底层原理的 CTO 问我：您感觉您适合这个行业吗？您简历里面列的项目我22岁的时候一个月就能做出来。说像博主这种“爱好者”非“工程师”，不适合做这项工作，之前有位雇员在他的公司做了一段时间回家开餐馆去了。虽然每天都有面试邀请，博主还是决定长期停止面试，回家 开 餐 馆。

#1 一般公司招聘流程

一定要明白了游戏规则才能玩好这个游戏。

  1. HR 筛选简历，推给技术
  2. 简历通过初选后技术面
  3. 过了技术面，人事面

#2 用产品思维写简历

终身学习博客看过很多关于怎么写简历的文章，其中收藏了几篇，分享给大家。

  • 用产品思维写好您的简历 —— 痕迹BLANK@简书
  • 牛逼程序员面试总结 - steed@博客园

其中《用产品思维写好您的简历》这篇文章让我非常受益，两年前的时候博主的简历放到招聘网站上面都没有主动上门的，而最近刚更新简历第一天开始每天都有面试。

基本信息 关于写不写毕业院校：企业如果很介意大专毕业的话，如果您没写大专毕业问了您一些算法、数据结构之类的问题加上对您大专学历的不屑您很大程度会无功而返，写上可以节省您的很多面试时间，其次如果您的实力真的很强写上是没有关系的。所以写不写您自己决定（博主现在是招聘网站中注明了，附件简历中没有注明，您要知道：HR 一般先看您的招聘网站中的 在线简历，然后把您的 附件简历 推给技术）。

项目介绍 一定要言简意赅，讲出重点。您要明白企业的招聘流程，看您简历的都是略懂技术的 HR 小姐姐，她们也是根据技术负责人的要求办事。

  • HR 小姐姐会根据职位要求和项目需求自己判断这个简历，比如现在公司的项目是实现一个 B2B商城 （这个梗比较冷了），如果看到简历有 “多层级分销商城”、“集成某某支付” 等关键词就可能会把您的简历推给技术。
    像这种实现了某某功能方式的介绍还是简单点比较好，因为简历最终还是要推给看“技术点”的技术
  • 有些负责人会根据职位需求给到 HR 几个要求，比如：“只要出现 Dreamweaver、h5、*** 的都不要”、“必须出现 Vue.js、React Native” 等的要求。
    这就要求您要写清楚您项目中使用的技术点，要专业，注意大小写，参见上面链接
  • 项目有链接的加上链接，建议放上短连接。因为企业面试的时候一般都会打印出简历来，如果只有锚文字打印出来是看不到链接的。所以简历想要深度展示的项目锚文字和短连接都要加上。

其他提示 除非您自己自带彩版简历，简历不要做成彩版，更不要加背景，因为企业通常会使用普通 A4 纸打印您的简历，如果上面一片一片的阴影会影响对您的印象。

#3 面试的技巧

首先还是要摆正自己的思想：公司招聘您是去做当下某个技术栈的某项目。终身学习博客从单个面试流程给出您一定建议：

  • 最重要：您与应聘岗位要求相关的项目经验。应聘 Java，结果 PHP 相关的项目经验总结的非常好，但是 Java 的却差强人意，这样对公司来说其实价值不大，公司需要的是一个做的好 Java 的人。
  • 您的其他亮点。除去 您与应聘岗位要求相关的项目经验 之外，如果您做过一些拿得出手的个人项目、开源项目或者参与维护迭代过的开源项目。像一个留言本之类换个人分分钟做出来的东西就不要放上去了。
  • 熟悉您的简历。您自己写的简历在介绍的时候一定不要表现出任何的不专业不流畅，如果您的组织能力不强或者比较紧张可以看着您的简历介绍（建议还是多看看自己写的简历）。
  • 不要随声附和。讲话的速度尽量放缓，说话之前要考虑一下面试官讲的话的意思，然后再回答。这样显得您思考过也不会因为紧张而出错。没人喜欢一直 “嗯嗯嗯”、“对对对”、“是是是“ 的人。

其他方面：

  • 前三次面试一定不要面您特别中意的企业。因为再优秀的人也需要预热和锻炼一下，一般第一次面试都是挂掉的，除非您的能力特别强，但不要高估自己的能力。
  • 基础不佳提前说明。像算法、数据结构、计算机网络这些基础不是一天两天就能拿起来的，如果这些基础不佳建议提前说明，避免问到时答不出来影响结果。
  • 面试题提前搞定。这是最基本的，因为工作中可能不会常常用到面试中提问到的问题，所以面试题和职位相关可能问到的问题一定要提前搞定，企业需要在短时间内确定您能胜任职位。

#4 善于总结

在面试过程中不断总结自己的优点和硬伤，在接下来的面试中不断地修正和提高。

#5 谈薪

关于”谈薪“这一块博主真的帮不了您了，百度一下您就知道。给出几条建议：

  • 先确定您的期望薪资。既不要膨胀也不要吃亏，先问自己低于多少您不会接受?

  • 说话速度尽量放缓。在人事给出 Offer 之后回答语速尽量慢，不要让人事找到您的底线。

    低于20k您去吗? 去
低于15k您去吗？去 ———— 这就是您大致底线
低于14k您去吗？不去 好，知道您的底线了
[如果您在担心下一个月的房租怎么交还是快速找到工作比较好]
  • 谈薪也是一轮面试。不要以为到了谈薪环节就一定通过了，不要狮子大开口，保持您的优雅与谦逊。

#6 祝您马到成功

如果您没有进入到心仪的企业也不要气馁，不论什么时候不要放弃学习，博主认为只要您的能力在那，价值终究也会在那。
